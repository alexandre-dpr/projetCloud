# projetCloud

## Lancement du projet
- Prod:
```bash
docker compose -f .\docker-compose-prod.yaml build
docker compose -f .\docker-compose-prod.yaml up -d
```

- Dev:
```bash
docker compose -f .\docker-compose-dev.yaml build
docker compose -f .\docker-compose-dev.yaml up -d
```
